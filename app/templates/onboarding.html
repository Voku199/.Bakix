Jak Bakix zachází s vašimi daty
  • ✓
    Přihlašovací údaje šifrované
    Bakix uloží vaše přihlašovací údaje v šifrované podobě (Fernet / AES-128) výhradně pro automatické obnovení přístupu po vypršení tokenu. Heslo v čitelné podobě není nikdy logováno ani sdíleno třetím stranám.
  • ✓
    Přihlašovací tokeny v session
    Bakix funguje stejně jako mobilní aplikace Bakalářů — uchovává dočasný token platný 8 hodin. Po odhlášení je smazán.
  • ✓
    Šifrované připojení
    Veškerá komunikace probíhá přes HTTPS — mezi vaším prohlížečem, Bakixem i serverem školy.
  • ✓
    Data přímo ze školy
    Bakix volá přímo Bakaláře API vaší školy — data neprocházejí žádnou třetí stranou.
